# PNG Crypto

Программа для **скрытия текстовых сообщений внутри PNG-картинок** с помощью **карты шума**. Сообщение шифруется и встраивается в младшие биты пикселей; визуально изображение почти не меняется.

## Как это работает

1. **Карта шума** (PNG 1024×1024) задаёт порядок, в котором по пикселям записываются биты сообщения.
2. Текст **шифруется** (потоковый XOR; при желании — с паролем).
3. Биты **встраиваются в LSB** каналов R, G, B выбранной картинки.
4. Для чтения нужны **та же карта шума** и **тот же пароль** (если использовался).

Две разные карты (`noise.png` и `noise_2.png`) — это **разные ключи**: сообщение, зашитое одной картой, другой не расшифровать.

## Требования

- Python 3.10+
- Windows / macOS / Linux (интерфейс на **tkinter**, обычно уже есть с Python)

## Установка

```bash
git clone https://github.com/AlexanderFilonov866/png-crypto.git
cd png-crypto
pip install -r requirements.txt
```

## Подготовка файлов

В репозитории **нет** картинок и карт шума — их нужно положить локально:

```
png-crypto/
├── карты шумов/          ← карты шума (PNG, 1024×1024)
│   ├── noise.png
│   └── noise_2.png
├── png 1024x1024/        ← исходные картинки-контейнеры (PNG, 1024×1024)
│   └── ...
├── main.py
└── ...
```

| Папка | Назначение |
|-------|------------|
| `карты шумов/` | Карты шума — определяют порядок встраивания |
| `png 1024x1024/` | Картинки, в которые можно прятать текст |

**Важно:** размер картинки и карты шума должен **совпадать** (например, оба 1024×1024).

## Запуск

```bash
python main.py
```

Откроется окно с двумя вкладками: **«Зашифровать»** и **«Расшифровать»**.

---

## Зашифровать сообщение

Вкладка **«Зашифровать»**:

1. **Картинка** — PNG, в который встроить текст (кнопка «…» или файл из `png 1024x1024/`).
2. **Карта шума** — выберите `noise.png` или `noise_2.png` (кнопки под полем или «…»).
3. **Сохранить как** — путь для нового PNG с сообщением (например, `output/secret.png`).
4. **Пароль** — необязательно; если задали при шифровании, понадобится при расшифровке.
5. Введите **сообщение** в текстовое поле.
6. Нажмите **«Встроить сообщение»**.

Готовый файл сохраняется как PNG. Его можно отправить кому угодно — без карты шума (и пароля) текст не прочитать.

---

## Расшифровать сообщение

Вкладка **«Расшифровать»**:

1. **PNG с данными** — файл, в который было встроено сообщение.
2. **Карта шума** — **та же**, что при шифровании.
3. **Пароль** — тот же, что при шифровании, или пусто, если пароль не использовали.
4. Нажмите **«Извлечь сообщение»**.

Текст появится в поле **«Расшифрованное сообщение»**.

## Ограничения и советы

| Правило | Почему |
|---------|--------|
| Сохраняйте только **PNG** | JPEG и другие форматы портят LSB-данные |
| Не пересжимайте результат | Потеряются встроенные биты |
| Одна карта шума на одно сообщение | Другая карта не подойдёт |
| Запоминайте пароль | Без него расшифровка не пройдёт |
| Размеры картинки и шума совпадают | Иначе программа выдаст ошибку |

Примерная вместимость на изображении 1024×1024 — **до ~390 КБ** текста (обычно этого более чем достаточно).
